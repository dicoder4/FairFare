FairFare – Microservices Demo

This UI talks to the API Gateway (port 3081), which routes requests to: user-service (users & groups), billing-service (bills) and splitter-service (splits).

1. Create User (user-service)

2. Create Group (user-service)

Use user IDs from above. Enter comma-separated user IDs.

3. Create Bill (billing-service → user-service)

This will call billing-service (via gateway), which internally calls user-service to validate groupId.

4. View Split (splitter-service → billing-service)

This calls splitter-service (via gateway). Splitter-service then calls billing-service to fetch bill details and calculates per-user share.
